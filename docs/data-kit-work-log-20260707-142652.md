# data-kit - Work Log

Version: v1.0.1 | Date: 20260707-142652 | Port: 4444

## Features
- Batch import/update/export products (WooCommerce)
- Translate product names/descriptions (DeepL)
- Sales reports and order summary
- Multi-tenant (X-Client-ID)
- Auto nonce refresh

## Endpoints
GET  /health
GET  /batch/export
POST /batch/import
POST /batch/update
POST /translate/products
POST /reports/sales
GET  /reports/summary

## Auth
WordPress cookie + nonce (auto-refresh via data-kit-refresh.php on WordPress).
Fallback: WooCommerce REST API Basic Auth.
