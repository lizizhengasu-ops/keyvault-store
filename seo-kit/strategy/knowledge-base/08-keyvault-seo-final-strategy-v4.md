## The Three Legs

### Leg 1: Backlinks (Trust) - Priority #1
New activation code sites face a trust barrier. Without external trust signals from backlinks, Google won't rank you regardless of content quality.

Methods (from Ahrefs 5 Ecommerce Link Building):
1. Product feedback - search best Windows license keys articles, email authors, offer free sample
2. Where to buy pages - search where to buy Windows keys, request listing
3. Unlinked mentions - monitor brand name mentions, request link addition
4. Directory submissions - Trustpilot, Sitejabber, Google Business Profile

### Leg 2: Content (Traffic) - Priority #2
2-3 articles per week, quality over quantity. Mix informational and transactional keywords.
SEO Kit auto-generates all SEO tags. publish first, optimize later.

Content plan: 12 articles in 6 weeks
Batch 1 (week 2-3): Win11 Pro vs Home / OEM vs Retail / Activate Win11 / Win10 vs Win11
Batch 2 (week 4-5): Best Win Keys 2026 / Bulk B2B guide / Office 2019 vs M365 / Install Office 2019
Batch 3 (week 6+): FAQ 15 questions / B2B page / Transfer license / Is OEM safe

### Leg 3: Platforms (Exposure) - Priority #3
Reddit r/WindowsHelp, Quora, YouTube for immediate traffic while SEO builds.
Links from these are mostly nofollow - helps with traffic, not directly with rankings.

## Tasks You Must Do Manually

### Week 1: Accounts + Setup
- Buy domain + SSL (Namecheap/GoDaddy/Cloudflare)
- Deploy KeyVault app to your hosting
- Create Google Search Console account (search.google.com/search-console)
- Create Google Analytics account (analytics.google.com)
- Create Google Business Profile (business.google.com)
- Create Trustpilot business account (trustpilot.com)
- Create Sitejabber business account (sitejabber.com)

### Week 1-4: Link Building (Manual, Critical)
- Search "best Windows license keys" on Google - find list articles, email authors
- Search "where to buy Windows keys" - find directory pages, request to be listed
- After each sale, email customer asking for Trustpilot review

### Ongoing Platform Presence
- Reddit: create account, help in r/WindowsHelp (dont spam)
- Quora: answer Windows/Office activation questions
- YouTube: optional - record activation tutorials
- WhatsApp Business: set up for B2B inquiries
- WeChat: set up for Chinese B2B market
